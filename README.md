# nuclear-project
**[ENG]**

This was a project in order to practice various functions within python.

It should be noted that all this code is in a fictitious setting and is for educational purposes only.

**[ESP]**

Este fue un proyecto para practicar varias funciones dentro de python.

Cabe señalar que todo este código se encuentra en un entorno ficticio y solo tiene fines educativos.

## It was practiced / Fue practicado
**[ENG]**
* Own functions (def)
* Menu inside terminal
* Login
* Native functions (if, loops, try, files, libraries)

**[ESP]**
* Funciones propias (def)
* Menú dentro de la terminal
* Inicio de sesión
* Funciones nativas (if, loops, try, files, librerias)

## Commands / Comandos
**[ENG]**
The commands are:

```
ayuda: Send a list of the commands in Spanish.
lanzar: The launch of the missiles to the country indicated below begins.
probar: Start system tests to see if everything is in order.
salir: Finish the session and the program.
```

**[ESP]**
Los comandos son:

```
ayuda: Muestra la lista de comandos.
lanzar: Inicia el protocolo de lanzamiento hacia el país entregado a continuación.
probar: Inicia las pruebas del sistema para ver si todo esta en orden.
salir: Cierra la sesión del usuario y finaliza el programa.
```
